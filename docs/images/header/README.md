# Images du Header

Ce dossier contient les 4 images qui tournent dans le header dynamique:

1. **satellite-meteo.jpg** - Image satellite météorologique (Europe Infrared)
2. **agriculture-femmes.jpg** - Femmes travaillant dans les champs
3. **agriculture-bananes.jpg** - Agriculture avec régime de bananes
4. **agriculture-mais.jpg** - Champ de maïs avec ciel nuageux

Les images doivent être au format JPG avec une résolution minimale de 1920x400px pour un affichage optimal en mode "cover".

## Transition dynamique

Les images changent toutes les 6 secondes avec:
- Transition fluide de 1 seconde (ease-in-out)
- Mode cover pour remplir tout l'espace
- Position centrée
- Overlay coloré semi-transparent pour lisibilité du texte
